Prescipto is a full-stack web application designed to simplify and digitize the management of medical prescriptions. The goal of this project is to create a seamless platform where users can manage, store, and access prescriptions efficiently.

The frontend is fully developed using React.js and Tailwind CSS, focusing on a modern, responsive, and user-friendly interface. It provides smooth navigation and clean UI components to enhance the user experience.

The backend is currently under development using Node.js, Express.js, and MongoDB, following a scalable and structured architecture. It will handle core functionalities such as data storage, API management, and user interactions.

🔧 Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js (in progress)
Database: MongoDB (in progress)
